# Smart CRM - Aplikacja dla Freelancerów
**Autor: Paweł Gabrysiak**

Smart CRM to nowoczesna aplikacja mobilna zbudowana w technologii Android, zaprojektowana by pomagać freelancerom w zarządzaniu ich bazą klientów, terminami oraz historią komunikacji.

## 📋 Podsumowanie Projektu
Aplikacja ewoluowała od podstawowego MVP do pełnoprawnego systemu zarządzania relacjami (CRM), wzbogaconego o automatyczne powiadomienia, inteligentne statystyki oraz personalizowany profil użytkownika.

## ✨ Kluczowe Funkcjonalności
- **Zarządzanie klientami**: Dodawanie, edycja, usuwanie i zdjęcia klientów (z trwałą galerią).
- **Inteligentne Terminy**: Grupowanie zadań według pilności (Zaległe, Dzisiaj, Nadchodzące) z możliwością odhaczania.
- **Statystyki**: Analiza aktywności w ujęciu tygodniowym, miesięcznym i całościowym.
- **Powiadomienia**: Systemowe przypomnienia o spóźnionych i zbliżających się terminach (High Priority).
- **Profil**: Możliwość ustawienia własnego imienia lub nazwy firmy.
- **Nowoczesna Nawigacja**: Dolny pasek nawigacyjny dla szybkiego dostępu do kluczowych funkcji.

## 🚀 Technologie
- **Jetpack Compose**: Nowoczesny interfejs użytkownika.
- **Room Database**: Lokalne i trwałe przechowywanie danych.
- **Dagger Hilt**: Wstrzykiwanie zależności.
- **WorkManager**: Zadania działające w tle.
- **Material 3**: Najnowszy standard wizualny od Google.
